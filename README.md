# TokiPonaIME
## トキポナ特化仮想キーボード
- 14音素のみのキーボード配列
- 5つの記号に対応（. , : ! ?）
- 入力途中で候補を表示
### 機能
1. スペース（kon）キー長押しで入力方法の選択可能
2. !キー長押しでキーボード配列を切り替え
	- アルファベット順配列
	- 出現頻度順配列
	- QWERTYライク配列
